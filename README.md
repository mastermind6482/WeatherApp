# WeatherApp

WeatherApp — это современное Android-приложение для отображения текущей погоды в выбранном городе. Проект реализован с использованием Jetpack Compose, архитектуры MVVM и современных библиотек для работы с сетью и UI.

## Основные возможности
- Поиск и отображение текущей погоды по названию города
- Актуальная информация: температура, облачность, влажность, давление и др.
- Современный дизайн с поддержкой светлой и тёмной темы
- Асинхронная загрузка данных с обработкой ошибок

## Архитектура и структура проекта
- **MainActivity** — точка входа, инициализация ViewModel и установка темы
- **WeatherViewModel** — слой ViewModel, отвечает за бизнес-логику, получение данных и управление состоянием UI
- **WeatherPage, WeatherDetails** — Composable-функции для отображения интерфейса и данных о погоде
- **api/** — модели данных (WeatherModel, Current, Location, Condition), интерфейс API (WeatherApi), синглтон RetrofitInstance, обработка сетевых ответов (NetworkResponse)
- **ui/theme/** — кастомизация темы, цветов и типографики приложения

## Используемые технологии и библиотеки
- **Kotlin** — основной язык разработки
- **Jetpack Compose** — декларативный UI
- **Retrofit2** — сетевые запросы
- **LiveData, ViewModel** — архитектурные компоненты Android
- **Coil** — загрузка изображений
- **Material3** — современные компоненты интерфейса
- **JUnit, Espresso** — тестирование

## Быстрый старт
1. Клонируйте репозиторий:
   ```sh
   git clone https://github.com/mastermind6482/WeatherApp.git
   ```
2. Откройте проект в Android Studio (Arctic Fox и выше)
3. Убедитесь, что установлен JDK 8+
4. Соберите и запустите приложение на устройстве или эмуляторе

## Настройка API ключа
API-ключ для доступа к погодному сервису хранится в файле `Constant.kt`. Для полноценной работы приложения замените значение `apiKey` на ваш собственный ключ.

## Требования
- Android 7.0 (API 24) и выше
- Интернет-соединение
